# <img src="https://api.shunnet.top/pic/shun.png" height=24> VisualIdentity


**VisualIdentity** 基于 .net9 与 [YoloDotNet](https://github.com/NickSwardh/YoloDotNet) 开发，实现了多模型管理，多任务单机运行，开箱即用


### 功能
1. 模型管理（增删改查）
2. 单机多个不同的识别任务
3. 简单应用

**YoloDotNet** 是一个速度极快、功能齐全的C#库，用于使用YOLOv5u-v12、YOLO World和YOLO-E模型进行实时对象检测、OBB、分割、分类、姿态估计和跟踪。


### 支持的版本:
```Yolov5u``` ```Yolov8``` ```Yolov9``` ```Yolov10``` ```Yolov11``` ```Yolov12``` ```Yolo-World``` ```YoloE```
